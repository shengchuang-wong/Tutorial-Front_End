Powerful Console

In Console

  1. inspect(element) will directly navigate to the element
  2. $0 will directly print out the element that is selected in inspect
  3. $$('div') you can get all element with div in the dom
  4. monitorEvents($$('textarea')[0]) you can get all the event with the dom element
  5. monitorEvents($$('textarea')[0], 'key') you can get all the KEY event with the dom element
  6. keys(document)
  7. values(document)
  8. copy('123') 123 will be directly copied into your clipboard
