Installation Procedure
======================
1. npm install -g vue-cli
2. vue init webpack-simple [project-name]
3. cd [project name]
4. npm install
5. npm run dev


Build Project
=============
1. npm run build

Tips
====
1. style is shared globaly. To make it explicit, use <style scoped></style>
